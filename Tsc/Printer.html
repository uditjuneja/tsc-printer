class Tsc::Printer

  • Tsc::Printer
  • Reference
  • Object

Overview

A Tsc::Printer allows for communicating with a TSC printer over the network.

printer = Tsc::Printer.new("1.2.3.4")
printer.connect do |p|
  p.cls
  p.direction(0, 0)
  p.text(121, 32, "0", 0, 30, 30, 0, "TEST")
  p.print(1)
end
